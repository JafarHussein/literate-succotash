Arithemetic operators

i) Increament operator

friends+=1;
print(friends)

ii)Decrement operator

friends-=1
print(friends)

iii)Multiplication

friends*=3
print(friends)

iv)Division
friends/=3
print(friends)

v)Exponents
friends **=3
print(friends)

vi)Modulus-This returns the remainder after a division operation

friends %= 3
print(friends)


In built Math functions

round()->This math functions performs rounding off

abs()->This returns the absolute value of a number, distance away from zero as a whole number

pow(x, y)->This raises the x to the power of y

max()->This returns the maximum value from a set of values

min()->This returns the minimum value from a set of values

**Importing the math module**

import math

math.pi->This returns the digits of pi

math.e->This returns the digits of the exponential constant

math.sqrt(value)->This return the square root of the value provided

math.ceil(value)->This will round up the provided value

math.floor(value)->This will always round a number down
