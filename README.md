# 🐍 Python Learning Journey

> A structured, project-driven path from absolute beginner to building real applications with Python — documented concept by concept.

---

## 📖 About This Repo

This repository tracks my complete Python learning journey. Every concept gets its own dedicated folder containing a `main.py` file where I write and experiment with code related to that topic. The goal is to build a solid foundation through hands-on practice and progressively more complex projects.

**Learning Source:** [Python Tutorial for Beginners – Bro Code](https://www.youtube.com/watch?v=XKHEtdqhLK8)  
**Total Topics:** 77 | **Projects:** 18 | **Duration:** ~11 hours of content

## 🚀 Getting Started

If you'd like to clone this repo and run any of the scripts yourself:

**Prerequisites**
- Python 3.10 or higher
- pip (comes bundled with Python)

**Clone the repo**
```bash
git clone https://github.com/your-username/python-learning-journey.git
cd python-learning-journey
```

**Run any script**
```bash
cd 28_number_guessing_game
python main.py
```

**For PyQt5 projects (topics 66–77), install the dependency first:**
```bash
pip install PyQt5
```
